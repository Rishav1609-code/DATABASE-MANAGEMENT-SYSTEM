A comprehensive, data-driven exam strategy report for the **B.Tech 6th Semester Database Management Systems (DBMS)** paper. This repository contains an interactive, visually structured guide built from a deep analysis of three years of Previous Year Questions (PYQs) and the official WBUT/MAKAUT organizer.

## 📌 Project Overview

Preparing for university exams often means sifting through disorganized notes and unstructured question banks. This project solves that by transforming raw PYQ data (2022-23, 2023-24, 2024-25) into a strategic, high-yield study plan. 

The main deliverable is a single-page, responsive HTML application (`index.html`) that provides an exact breakdown of what to study, how questions are weighted, and what is most likely to appear in the upcoming exam.

## ✨ Key Features

- **📊 Topic Frequency Heatmap:** Visualizes the most repeated topics (e.g., ACID properties, B-Tree insertions, 2PL) to help prioritize high-yield areas.
- **🔮 Predicted Questions:** Probability-based predictions for Group A (VSA), Group B (SA), and Group C (LA) grounded in historical rotation patterns.
- **🗺️ Chapter-Wise Roadmap:** A weighted breakdown of all 9 DBMS chapters, highlighting the specific concepts that carry the most marks.
- **📅 5-Day Battle Plan:** A structured, day-by-day study schedule designed to cover the entire syllabus efficiently before the exam.
- **🧠 Must-Know Formulae & Rules:** Quick-reference cards for essential rules like Armstrong's Axioms, 2PL phases, and Normalization progressions.
- **💡 Exam Hall Tactics:** Actionable tips on answering numericals, drawing ER diagrams, and maximizing partial marking.

## 🛠️ Tech Stack

This project is built using pure, lightweight front-end technologies for maximum speed and accessibility without the need for a server:
- **HTML5** (Semantic structuring)
- **CSS3** (Custom variables, responsive CSS grid/flexbox layouts, modern typography via Google Fonts)

## 🚀 How to Use

1. Clone or download this repository to your local machine.
2. Navigate to the project directory.
3. Open the `index.html` file in your preferred web browser (e.g., Brave, Chrome, Firefox).
4. Scroll through the sections to analyze the PYQ data, review the 5-day study plan, and drill down into the chapter roadmaps.
